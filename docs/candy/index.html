Optional CSS

Candy Components

Production-ready components built on Semantastic design system primitives.

candy.css is an optional, higher-level CSS layer that provides 15 components implemented as custom elements — replacing verbose inline styles with clean, semantic markup.

Note: This file is completely optional. The Semantastic system works perfectly without it. Candy.css is the only CSS file in the system that imports dependencies.

CSS Background Patterns

Background Patterns

Pure CSS background patterns applied via utility classes. No images, no JavaScript — just CSS gradients and custom properties. Configure size, color, and density inline.

All patterns use background-repeat: repeat and background-position: center by default. Apply to any element via a pattern class.

Usage

<!-- Basic -->
                <div class="pattern-dots"></div>

                <!-- Configured -->
                <div class="pattern-grid"
                style="--pattern-size: 30px;
                --pattern-fill: var(--color-primary);
                --pattern-width: 2px;">
                </div>

Shared Properties

Most patterns accept these custom properties (see individual notes for exceptions).

--pattern-size varies per pattern --pattern-fill #888 --pattern-width 1px (line patterns)

Colorize Utility

One Color In, Full Palette Out! The data-colorize utility automatically generates a complete, harmonious color palette from a single color input. Perfect for rapid theming and prototyping.

How it works: Provide a single OKLCH color via --colorize, and the utility generates 10 coordinated properties using color-mix() — surfaces, text colors, accents, and borders.

Generated Properties

From one color input, you get:

Surfaces: --surface, --surface-hover, --surface-active
                Text: --text, --text-heading, --text-on-accent
                Accents: --accent, --accent-emphasis
                Borders: --border, --border-strong

Example 4: Interactive Button States

      background: var(--surface);

      border-radius: 6px;
      cursor: pointer;
      font-size: 1rem;
      transition: background 0.2s;
    " onmouseover="this.style.background='var(--surface-hover)'" onmouseout="this.style.background='var(--surface)'"
                onmousedown="this.style.background='var(--surface-active)'"
                onmouseup="this.style.background='var(--surface-hover)'">
