Foundation Layer

Design Tokens

The foundational design values that power the entire Semantastic system.

Every system should have tokens. Tokens are units of common values to make sure there is a sense of conisetency to the site. Large text is the same size across the site, spacing is consistent. Make your users brain happy by allowing them to understand where they should expect content to be sitting in relation to one another.

This is a minimal set of tokens to build a website. In practice you'll either want to codify your own opinions or find one of the many excellent existing resources such as Open Props for a more thorough experience.

All tokens are defined as CSS custom properties in tokens.css. They cascade throughout the system and can be overridden at any level.

Measure (Line Length)

Control text length with explicit ch counts

--measure-short (45ch)

Good for narrow columns and sidebars. Creates a more compact reading experience.

--measure (60ch)

The default. Ideal for body text and article content. This is considered the optimal line length for comfortable reading.

--measure-long (75ch)

For wider content areas where text is supplemented by images or code blocks. Still readable but gives more breathing room for mixed content.
