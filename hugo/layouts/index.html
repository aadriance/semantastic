Semantastic gives you a focused toolkit — each file is independent so you include exactly what your project needs. No utility soup, no framework lock-in.

tokens.css

Design Tokens

Spacing scale, type scale, shadows, border radius, timing, and measure — all as :root custom properties.

 colors.css

Color System

OKLCH semantic aliases with automatic dark mode. One variable override to retheme the whole site.

 layouts.css

Layout Primitives

12 composable custom elements — stack, grid, sidebar, cover, reel, and more. No media queries.

 components.css

Pure CSS Components

Tabs, flyout drawers, and tooltips built with native HTML features. Zero JavaScript.

 candy.css

Candy

Cards, badges, notices, pattern backgrounds, and a colorize mixin for on-the-fly theming.

Usage

There is no CDN, no minified bundle. If you would like to use Semantastic I recommend you copy the files yourself and start tweaking to meet your needs. I don't intend for anyone to take a hard dependence on Semantastic. I wanted to make a easy starting point for me to copy down thing I know I want to use often.
If you can use this as is great. If I've inspired you to do some of your own CSS, even better. Go forth and make standards compliant UI! {{ end }}
