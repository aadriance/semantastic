Layout Primatives

Custom tags inspired by Every Layout.

A collection of custom tags for layout elements to control the flow of your page. Based on Every Layout the goal is to accept that your website is not a poster and instead build primitives that make sure your sites content flows correctly regardless of device size.

Custom tags are used because div soup is bad. HTML allows you to name your own tags, so why not leverage it? often you'll see class="foo" instead. But what if Bar is a class as well? class="foo bar" may indeed be nonsensical. As such enforcing the layouts as distinct elements creates the nicest experience in my opinion.

I won't reiterate all the content of Every Layout. So go get Every Layout for real. Totally worth it.
